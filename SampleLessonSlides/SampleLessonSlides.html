Creating a mapping

In ggplot2, matching dataset columns to plot components is called “mapping.”

Plot components are called visual properties or “aesthetics.”

 

Examples:

  • x
  • y
  • color
  • fill
  • size
  • alpha (transparency)

Step 1: Make a pair

<aesthetic> = <column name>

 

Examples:

 

  • x = displ
  • y = hwy

Step 2: Combine pairs into list

x = displ, y = hwy

Step 3: Wrap the list with aes()

aes(<list of mapping pairs>)

 

aes(x = displ, y = hwy)

Step 4: Insert list into geometry layer

geom_...(mapping = aes(...))

 

geom_point(mapping = aes(x = displ, y = hwy))
